\begin{frame}
  \begin{itemize}
  \item \textbf{data structure} is a systematic way of organizing and accessing dat;
  \item \textbf{algorithm} is a step-by-step procedure for performing some task in a finite amout of time.
  \end{itemize}

  The primary analysis tool involves \blue{characterizing the running time of algorithms and data structure algorithms, with space usage also being of interest.}
\end{frame}

\begin{frame}
  Running time is a natural measure of ``goodness'', since time is a precious time - computer solutions should run as fast as possible.

  \begin{itemize}
  \item increases with the input size
  \item be affected by the hardware environment (e.g., the processor, clock rate, memory, disk) and software environment (e.g., the operating system, programming language). 
  \end{itemize}
\end{frame}
